## Design Principles

### 1. Sovereign Aesthetic (Zinc-950 Dark Mode)
```css
/* Core palette */
--bg: #09090b;          /* zinc-950 */
--surface: #18181b;     /* zinc-900 */
--border: #27272a;      /* zinc-800 */
--text: #fafafa;        /* zinc-50 */
--muted: #a1a1aa;       /* zinc-400 */
--accent: #8b5cf6;      /* violet-500 */
```

### 2. Motion Language
```typescript
// From @wlf/ui/motion
EASING.sovereign = [0.23, 1, 0.32, 1]  // Signature smooth
DURATION.normal = 0.4                    // Standard transition
SPRING.snappy = { stiffness: 400, damping: 30 }  // Interactive
```

### 3. Vertical Rhythm (8px Grid)
```css
/* Section spacing */
.section-py-sm: py-16    /* 64px - Compact */
.section-py-md: py-24    /* 96px - Standard */
.section-py-lg: py-32    /* 128px - Breathe */
```

### 4. Container Sizes
```css
/* Max widths */
.container-sm: max-w-4xl   /* 896px - Content */
.container-md: max-w-5xl   /* 1024px - Standard */
.container-lg: max-w-6xl   /* 1152px - Wide */
.container-xl: max-w-7xl   /* 1280px - Full */
```

## AI Composition Pattern

When AI creates a landing page, it should be able to say:

```json
{
  "title": "Victor's Landing Page",
  "slug": "victor",
  "hero": {
    "type": "highImpact",
    "badge": "New for 2026",
    "headline": "Build Wealth Through Real Estate",
    "subheadline": "Join 500+ investors...",
    "primaryCTA": { "label": "Get Started", "url": "/apply" }
  },
  "layout": [
    { "blockType": "socialProof", "logos": [...] },
    { "blockType": "featureGrid", "features": [...] },
    { "blockType": "testimonials", "items": [...] },
    { "blockType": "faq", "items": [...] },
    { "blockType": "cta", "headline": "Ready?" }
  ]
}
```

And Payload stores it. Frontend renders it. Done.
